# Game Site Keyword Opportunities - 2026-06-20

Scope: first-pass market and keyword direction for a low-cost Cloudflare game SEO site.

Skills applied:

- `site-keyword-research`: keyword layering, SERP competition scoring, Top 3 direction.
- `html5-game-radar`: HTML5/browser-game opportunity logic.
- `roblox-site-architect`: Roblox game tools architecture, codes/tier-list/calculator defaults, no fabricated game data.

## Executive Decision

Recommendation: start with a Roblox SEO tools/content cluster, not a generic HTML5 game portal.

The best first cluster is `99 Nights in the Forest`, followed by `Grow a Garden 2` as the second cluster.

Why:

- Roblox codes/tools have clear repeat demand and frequent freshness hooks.
- Cloudflare deployment fits static/dynamic SEO pages well.
- 7Deer skills are strongest for Roblox codes, tier lists, calculators, and game data pages.
- Generic browser-game and unblocked-game portals have broad traffic, but the SERP is crowded by large aggregators and school/unblocked sites. They require a game library and moderation/safety work before they become useful.

## Top 3 Opportunities

### 1. 99 Nights in the Forest Codes + Survival Hub

Priority: first build.

Reason:

- It appears in current Roblox top-playing-now results.
- Codes pages are actively updated in June 2026 by major sites, proving demand.
- The game also has durable guide demand: classes, survival tips, entities, and wiki-style lookup pages.
- Unlike Grow a Garden 2 calculators, the exact-match tool SERP is not yet as saturated with dedicated calculator domains.

Initial pages:

- `/roblox/99-nights-in-the-forest/codes`
- `/roblox/99-nights-in-the-forest/classes`
- `/roblox/99-nights-in-the-forest/class-tier-list`
- `/roblox/99-nights-in-the-forest/entities`
- `/roblox/99-nights-in-the-forest/survival-guide`

Data requirements:

- Active/expired codes with source and checked date.
- Class list with abilities/perks from verified wiki or in-game/creator sources.
- Entity list and survival mechanics from verified wiki and PC Gamer/Fandom coverage.

### 2. Grow a Garden 2 Codes + Tier List

Priority: second build.

Reason:

- Massive franchise demand and fresh June 2026 coverage.
- Good topical expansion into plants, gears, pets, values, stock, and calculators.
- Competition is already heavy for calculators and value lists, so the first Grow a Garden 2 page should target freshness and breadth before attempting a tool.

Initial pages:

- `/roblox/grow-a-garden-2/codes`
- `/roblox/grow-a-garden-2/plant-tier-list`
- `/roblox/grow-a-garden-2/gear-tier-list`
- `/roblox/grow-a-garden-2/best-plants`
- `/roblox/grow-a-garden-2/stock-notifier` only if we can verify a reliable data source.

Data requirements:

- Codes from at least two sources.
- Plant/gear tiers from current June 2026 guides.
- No calculator until formulas and current values are verified.

### 3. Multi-Game Roblox Codes Hub

Priority: supporting page after the first two clusters.

Reason:

- Useful internal linking layer.
- Helps capture broad `Roblox codes June 2026` type queries.
- Should not be the whole site at first because Beebom, PC Gamer, Pocket Tactics, GamesRadar, and similar domains are strong competitors.

Initial page:

- `/roblox/codes`

Structure:

- A table of games with checked date, active-code count, latest reward, and links to each game-specific page.
- Start with 99 Nights in the Forest, Grow a Garden 2, Fisch, Blue Lock Rivals, and Steal a Brainrot.

## What Not To Do First

Do not start by cloning a broad CrazyGames-style portal. It requires a large game catalog, licensing/embedding decisions, and quality moderation before it is useful.

Do not start with `unblocked games` as the main angle. It has search demand, but the SERP is noisy, brand trust is lower, and the topic can create school-network/safety baggage.

Do not build Roblox calculators with invented formulas. Per `roblox-site-architect`, calculators require real data sources and a preflight checklist. Codes and guide pages can ship earlier.
