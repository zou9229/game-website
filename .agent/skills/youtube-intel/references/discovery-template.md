## 报告示例（完整案例）

### 需求输入
> "帮我分析 AI 视频工具这个类目"

### 第一步：需求分析输出
```
【需求分析】
原始输入："AI 视频工具"
分析结果：
  - 类目粒度：中等（比"AI 工具"细，但还有分）
  - 已拆解为 3 个子分类：
    1. AI 视频生成工具（搜索词：AI video generator tools 2025）
    2. AI 视频编辑工具（搜索词：AI video editing tools 2025）
    3. AI 虚拟形象/数字人（搜索词：AI avatar tools 2025）
```

```
【搜索策略】

  [1] AI 视频生成工具
```

## 报告阅读指南

**置信度说明：**
- 🟢 高：直接抓取 YouTube 页面，数据可信
- 🟡 中：部分数据来自 YouTube 模糊化估算
- 🔴 低：数据来源混杂，或结果太少无法判断

**竞争度说明：**
- 🔴 高：头部已占位，需要强差异化才能突围
- 🟡 中：有空间但需要找准角度
- 🟢 低：新兴市场，机会窗口仍在
- ⚪ 空白：数据不足，无法判断
