## Ground Rules

- Do not target broad `Roblox` as a primary keyword. It is navigational and far
  too competitive for a new domain.
- Use Semrush Volume/KD as a filter, not as the only decision. A keyword with KD
  50 can still support a cluster page, but it should not be the only bet.
- Favor queries where the SERP has mixed results: wiki pages, YouTube videos,
  Reddit, older guides, or pages with missing source trails.
- Build pages only when Quest Codes can add something concrete: source checks,
  current status, structured data, internal links, and update policy.
- Track unverified community claims separately from confirmed guide data.
