## Purpose

Quest Codes is a fan-made Roblox codes and guide site. Our goal is to publish practical Roblox game references that players can use quickly while making the source status clear.

We prioritize:

- Working code lists with active, special, and expired codes separated.
- Guide pages that answer one player intent clearly.
- Update notes that distinguish source-checked changes from metadata-only checks.
- Internal links that help players move from codes to related routes, classes, crafting, gems, maps, badges, and survival decisions.

## Source Standards

Published facts should come from one or more of these source types:

- Official Roblox game pages, public developer posts, or official social updates.
- Trusted gaming publications or established guide sites.
- Public community sources that can be checked without relying on private screenshots or unverifiable claims.
- Direct in-game testing when available.

When sources disagree, Quest Codes labels the conflict instead of silently merging the claims.

## Code Status Rules

Code pages separate code status into active, special, expired, and conflicting entries.

- Active codes are listed when a source trail supports the current status.
- Special codes are codes that may depend on events, platforms, timing, chat entry, server state, or conflicting source notes.
- Expired codes are kept when they help players avoid retesting old rewards.
- Rumored codes are not promoted as working codes without review.

Code status can change quickly after Roblox events, milestone rewards, apology drops, and developer updates. Players should retry codes carefully.

## Guide Claim Rules

Quest Codes avoids publishing invented stats, fake drop rates, hidden formulas, or unverified patch details. If a page cannot support a claim with a reasonable source trail, the claim stays out of the main factual copy.

Guide pages may include cautious recommendations when the reasoning is visible, but they should not present guesses as confirmed game data.

## Images And Media

Quest Codes uses game-related images only for identification, commentary, navigation, or explanation. Roblox names, game names, thumbnails, and related marks belong to their respective owners.

When a screenshot, thumbnail, or video embed is useful, it should support a specific player task rather than serve as decoration.

## Update Cadence

The codes page receives the highest refresh priority because codes can expire or change faster than guide pages.

Typical review triggers include:

- A major Roblox game update.
- A public developer announcement.
- A trusted source changing its code list.
- A new event, apology reward, milestone reward, or limited-time drop.
- Search data showing a new guide topic with enough demand to justify a full page.

## Advertising And Affiliate Disclosure

Quest Codes may add advertising or affiliate links in the future. If that happens, monetized links or ad placements should not control code status, guide conclusions, or source labels.
