# Site Keyword Research

> 整站关键词研究与深度挖掘 — 递归式关键词树扩展版

## 功能

输入一个网站域名，自动完成从关键词挖掘到竞争度分析的全流程，输出可直接指导 SEO 行动的策略报告。

**核心能力：递归式关键词树扩展** — 不是一次扩展就停，而是持续分叉挖掘 Google 联想词网络，直到词库自然收敛或达到 100 词上限。

---

## 工作原理（三阶段）

```
阶段一：递归关键词树扩展
    种子词 → Google 搜索 → 提取 Related Searches + PASF + PAA 问题词
    → 每个新词继续搜索 → 持续分叉
    → 直到词库收敛（队列空）或达到 100 词上限

阶段二：分层筛选（20词）
    去重 + 去无关词 → 四层分级（核心/中尾/长尾/问题）
    → 选 20 个最有价值的词进详细分析

阶段三：10词 SERP 分析 → 3词定方向
    打开 Google SERP，记录广告主/权重站/Featured Snippet/内容深度
    → 竞争度打分 → 给出 Top 3 词的具体操作建议
```

## 关键词来源标注

每个候选词都标注了来源和可信度：

| 标签 | 含义 | 可信度 |
|------|------|-------|
| `[PASF]` | Google「People Also Search For」真实推荐 | ⭐⭐⭐⭐⭐ |
| `[RS]` | Google Related Searches 联想词 | ⭐⭐⭐⭐ |
| `[Q]` | Google「People Also Ask」问题词 | ⭐⭐⭐⭐ |
| `[AI-主题]` | AI 根据站点主题扩展生成，需人工验证 | ⭐⭐⭐ |
| `[竞品]` | 从 SERP 竞争者内容提取 | ⭐⭐⭐⭐ |

每个词还标注了**扩展深度**（depth=1/2/3+），depth=1 最相关，depth=3+ 可能发现意外的低竞争词。

## 竞争度打分

对每个 SERP 分析的词，从 5 个维度打分（1-5分）：

| 维度 | 1分 | 3分 | 5分 |
|------|-----|-----|-----|
| 广告主 | 0 | 3-5个 | 8个以上 |
| 高权重站 | 0 | 2-3个 | 5个以上 |
| Featured Snippet | 无 | 1个 | 2个以上 |
| 视频结果 | 无 | 1-2个 | 3个以上 |
| 内容深度 | 全薄页 | 混合 | 全深度 |

- 🟢 **5-10 分**：低竞争，立即行动
- 🟡 **11-17 分**：中竞争，可切入
- 🔴 **18-25 分**：高竞争，观望迂回

## 适用场景

- **新站点 SEO 启动**：找到真正有机会的关键词，避免在红海词上浪费内容
- **竞品调研**：了解竞争对手在哪些词上占位，发现漏斗顶端的空白
- **内容规划**：基于关键词分层制定 1-3 个月的内容日历
- **外链策略**：从 Top 关键词的 SERP 分析中找到外链机会

## 为什么用递归扩展？

传统关键词研究通常：
1. 种子词 → 扩一轮 → 结束 → 选 10 个词分析

问题是：只扩一轮会遗漏大量相关词。

递归扩展的优势：
- 从种子词直接扩展的词 → depth=1（最相关）
- 从 depth=1 词再扩展的词 → depth=2（覆盖更广，可能发现长尾）
- 多级扩展后，depth=3+ 可能发现**低竞争的意外机会**（竞品没覆盖到的角落）

## 反爬说明

本 skill 使用真实的 browser 访问 Google 搜索结果，需要遵守：

1. **每次搜索后等待 3-5 秒**（给 JS 渲染时间）
2. **连续搜索间隔随机 2-4 秒**（防止固定频率被识别）
3. **遇到 CAPTCHA 最多重试 2 次**（仍失败则降级到 web_search 并在报告中标注）

建议：使用美国节点的 browser 可获得最准确的 SERP 数据。

## 与旧版区别

本版本（v2）是**递归扩展版**，与旧版的区别：

| | 旧版（v1）| 新版（v2）|
|---|---|---|
| 扩展方式 | 一次扩展 | 递归持续分叉 |
| 词库规模 | 30+ 词 | 80-100 词 |
| 来源标注 | 仅来源类型 | 来源类型 + 扩展深度 |
| 扩展深度 | 无 | depth=1/2/3+ |
| 推荐词来源 | 无 | PASF 优先 |
